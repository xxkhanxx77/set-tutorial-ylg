What you'll learn

Two market-neutral strategies on real Thai-derivatives data — the same idea (profit from a price gap, not from the market going up or down) applied to two different pairs.

Part A · statistical

The USD dollar spread

Binance TH USDTTHB vs TFEX USD futures USDM26 — the same dollar priced in baht in two places. It usually reverts, but nothing forces it.

Part B · convergence

The gold mini/big arbitrage

TFEX MGO (mini) vs GO (big) gold futures: 10 minis = 1 big, both cash-settling on the same gold price. The gap must close at expiry.

Every notebook comes with its results already saved. Open any .html, .ipynb, or .md and read top to bottom — all charts and tables are there. You do not need to run anything, and you don't need any data, account, or API key to follow along.

Setup with uv

Only needed if you want to re-run a notebook. Just reading the course needs none of this — the saved outputs are complete.

curl -LsSf https://astral.sh/uv/install.sh | sh   # install uv (once)
uv sync                                            # build the environment from uv.lock
uv run jupyter lab                                 # open any step and run it

Re-running needs credentials — copy .env.example to .env and fill in:

  • Settrade API keys — for the TFEX candle reference (steps 1, 3) and the gold candles (step 6).
  • DATABASE_URL — a PostgreSQL database the data logger has filled with captured bid/ask (steps 2, 4, 5).

What this is (and isn't)

✓ Is

Honest, worked examples of market-neutral strategies on real Thai-market data, with every cost, margin, and assumption stated out loud.

✗ Isn't

Trading advice or an order-placing bot. No notebook places an order, and the numbers describe specific (often quiet) data windows — treat the percentages as a method to learn, not a promised yield.
